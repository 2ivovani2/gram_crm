# Руководство пользователя бота

---

## Для воркеров

### Начало работы

1. Найдите бота в Telegram и нажмите **Start** (или отправьте `/start`).
2. Бот выведет приветствие. Для получения доступа нужен **invite key** — запросите его у администратора.
3. Нажмите кнопку **«Ввести invite key»**, введите ключ в чат.
4. После успешной активации вы получите полный доступ к личному кабинету.

> Ключ не чувствителен к регистру. Если ключ не принимается — проверьте срок действия или обратитесь к администратору.

---

### Главное меню

После активации `/start` открывает главное меню с кнопками:

| Кнопка | Действие |
|--------|----------|
| 👤 Профиль | Личный кабинет со статистикой |
| 👥 Рефералы | Список ваших рефералов и реф. ссылка |
| 📊 Статистика | Статистика по задачам за 7 дней |

---

### Профиль

Нажмите **«👤 Профиль»** чтобы увидеть:

- Ваш Telegram ID
- Имя и статус аккаунта
- Баланс (в рублях)
- Количество рефералов и привлечённых людей
- Рабочую ссылку (если назначена администратором)
- Вашу реферальную ссылку
- Статистику за сегодня: заявки / выполнено / % выполнения
- Дату активации

**Статусы аккаунта:**
- ⏳ Ожидает — аккаунт создан, ещё не активирован
- ✅ Активен — полный доступ
- ⛔ Неактивен — доступ временно приостановлен
- 🚫 Заблокирован — обратитесь к администратору

---

### Рефералы

Нажмите **«👥 Рефералы»**:

- Отображается ваша реферальная ссылка вида `t.me/<botname>?start=ref_<token>`
- Список последних 10 привлечённых пользователей
- Ваша ставка с заработка реферала (% устанавливает администратор)

**Как работает реферальная программа:**
1. Скопируйте вашу реф. ссылку.
2. Поделитесь ею с новым пользователем.
3. Когда человек перейдёт по ссылке и запустит бота — он привяжется к вам как реферал.
4. С каждого заработка реферала вам начисляется указанный %.

---

### Статистика

Нажмите **«📊 Статистика»** — отображается статистика за последние 7 дней:
- Дата
- Заявок отправлено / Выполнено
- Процент выполнения

---

### Реферальная ссылка

Ваша уникальная реферальная ссылка доступна в разделах «Профиль» и «Рефералы».  
Формат: `https://t.me/<имя_бота>?start=ref_<токен>`

Привязка реферала происходит однократно при первом `/start` по вашей ссылке.

## Для администраторов (в боте)

### Доступ к панели

Команда `/admin` открывает панель администратора.  
Если вы зашли через `/start` — бот автоматически перенаправит вас в панель (если ваш аккаунт имеет роль администратора).

---

### Главное меню администратора

| Раздел | Назначение |
|--------|------------|
| 👥 Пользователи | Список пользователей, карточки, управление статусами |
| 🔑 Invite Keys | Создание и управление ключами приглашения |
| 📢 Рассылки | Создание и запуск рассылок |
| ⚙️ Настройки | Реферальная ставка |

---

### Пользователи

**Список:** отображаются все пользователи с пагинацией (5 на страницу).

**Поиск:** кнопка «🔍 Поиск» — введите Telegram ID, имя или @username.

**Карточка пользователя** показывает:
- Telegram ID, имя, username
- Роль и статус
- Баланс, рефералы, привлечено людей
- Рабочую ссылку (если задана)
- Дату регистрации и последней активности

**Действия на карточке:**
- **«🔄 Статус»** — изменить статус: Активен / Неактивен / Заблокирован
- **«🔗 Установить рабочую ссылку»** — задать URL для воркера (https://...)  
  Отправьте «-» чтобы удалить ссылку.
- **«👤 Привлечено людей»** — вручную задать число привлечённых людей (целое ≥ 0)

---

### Invite Keys

**Список ключей** показывает все созданные ключи с пагинацией.

**Карточка ключа** содержит:
- Сам ключ (копируемый)
- Метку (внутреннее название)
- Статус (активен / исчерпан / истёк / деактивирован)
- Лимит и текущее количество использований
- Дату истечения

**Создание ключа** — 3 шага:
1. Введите **метку** (внутреннее название, не видно пользователям). Отправьте «-» если не нужна.
2. Введите **лимит использований** (например: `10`). Отправьте «0» или «-» для неограниченного.
3. Введите **срок действия** в формате `ДД.ММ.ГГГГ`. Отправьте «-» для бессрочного.

После создания ключ можно скопировать и передать воркеру.

**Активация/деактивация:** кнопка «Активировать/Деактивировать» переключает статус ключа без удаления.

**Лог активаций:** кнопка «📋 Активации» — кто и когда использовал ключ.

---

### Рассылки

**Создание рассылки** — 3 шага:
1. Введите **внутреннее название** (не отправляется пользователям).
2. Введите **текст сообщения**.  
   Поддерживается HTML-форматирование: `<b>`, `<i>`, `<code>`, `<a href="...">`.
3. Выберите **аудиторию**:
   - 📢 Всем — всем пользователям в базе
   - ✅ Активным — только активным пользователям
   - 🔑 По ключу — пользователям, активированным через invite key

**Процесс запуска рассылки:**
1. После создания рассылка находится в статусе **«Черновик»**.
2. Нажмите **«✅ Подтвердить»** — статус меняется на «Подтверждена».
3. Нажмите **«🚀 Запустить»** → подтвердите запуск (отобразится кол-во получателей).
4. Рассылка переходит в статус **«Выполняется»**, затем **«Завершена»**.

> После запуска остановить рассылку нельзя.

**Логи доставки:** кнопка «📋 Логи» — статус доставки каждому получателю:
- ✅ Отправлено
- ❌ Ошибка
- 🚫 Пользователь заблокировал бота

---

### Настройки

**«⚙️ Настройки»** → **«✏️ Изменить ставку»**:
- Введите реферальную ставку в процентах (например: `5` или `2.5`)
- `0` — отключить реферальную программу
- Изменения применяются немедленно ко всем пользователям

---

### Общие команды

| Команда | Действие |
|---------|----------|
| `/start` | Главное меню (для роли перенаправляет: воркер → меню воркера, админ → панель) |
| `/admin` | Открыть панель администратора (только для администраторов) |
