SpamControl: Админка

Это главный инструмент управления воркерами. Здесь вы добавляете людей, выдаёте им рабочие ссылки, устанавливаете ставки и смотрите статистику.

🗂️ Структура системы

👤
Пользователи
Все люди в системе. У каждого есть роль (Воркер / Куратор / Администратор) и статус (активен / неактивен / заблокирован).
🔗
Рабочие ссылки
Каждому воркеру выдаётся рабочая ссылка. По ней считается, сколько подписчиков он привлёк. Назначение происходит через панель «Клиенты».
🏢
Клиенты и ссылки
Панель /stats/clients/ — управление клиентами и их ссылками. Авто-назначение воркеров, ручное назначение, редактирование заявок.
📢
Рассылки
Массовые сообщения всем или отдельным группам пользователей через Telegram.

➕ Как добавить нового воркера

Воркеры вступают в систему через заявку — никаких инвайт-ключей не нужно.

1
Воркер пишет /start боту
Бот показывает кнопку «Подать заявку». Воркер нажимает — заявка уходит всем администраторам с кнопками «Одобрить» / «Отклонить».
2
Вы одобряете заявку
Нажмите «✅ Одобрить» в уведомлении от бота. Воркер сразу получит сообщение с подтверждением и увидит своё главное меню.
3
Установите ставку
В Telegram-боте зайдите в управление воркером → «Изменить ставку». Укажите personal_rate — сколько рублей воркер зарабатывает за каждого привлечённого подписчика.
4
Назначьте рабочую ссылку
Перейдите в Статистика → Клиенты. Найдите нужную активную ссылку — система авто-назначит воркера с наименьшей нагрузкой. Или назначьте вручную через выпадающий список в той же таблице.

🏢 Управление клиентами и ссылками

Панель Статистика → Клиенты — центральное место для работы с клиентами, их ссылками и назначением воркеров.

➕
Добавить ссылку клиенту
В карточке клиента нажмите «Добавить ссылку», введите URL. После добавления система автоматически назначит свободного воркера — он получит уведомление в бот.
⚠️
Ссылка без воркера
Если свободных воркеров нет — ссылка подсветится оранжевым. Под ней появится выпадающий список для ручного назначения воркера.
✏️
Редактировать заявки
В столбце «Заявок» рядом с каждой ссылкой есть поле ввода. Введите актуальное число и нажмите ✓ — баланс воркера пересчитается сразу.
💰
Финансовая сводка
В нижней части страницы — итоги по каждому клиенту: сколько заработал клиент, выплачено воркерам и рефералам, и чистая прибыль.
💡
Авто-назначение и уведомления
При добавлении ссылки система выбирает воркера с наименьшим числом активных ссылок. Воркер сразу получает сообщение в бот: «Вам назначена новая ссылка — Клиент: X / URL: …». Назначать вручную нужно только если авто-выбор не нашёл подходящего.

👤 Управление пользователем

Откройте Пользователи → найдите нужного через поиск → нажмите на имя.

Основные поля

ПолеЧто означает
roleРоль: worker — обычный воркер, curator — управляет другими воркерами, admin — полный доступ
statusactive — работает; inactive — временно отключён; banned — заблокирован навсегда
work_urlРабочая ссылка (денормализованный кэш — меняйте через бота или через кнопку «Заменить ссылку»)
attracted_countЧисло привлечённых подписчиков по активной ссылке (кэш)
personal_rateСтавка за одного подписчика (руб/чел) — личный заработок воркера
referral_rateСтавка за подписчиков, привлечённых рефералами этого воркера
balanceТекущий баланс (пересчитывается автоматически при изменении данных)
⚠️
Рабочие ссылки — меняйте только через бота
Когда ссылка меняется, старый счётчик подписчиков замораживается (история сохраняется), новая ссылка начинает считать с нуля. Это важно для корректного расчёта баланса. Используйте функцию «Заменить ссылку» в Telegram-боте, а не прямое редактирование поля в этой панели.

📊 Просмотр статистики

Основная статистика видна прямо в карточке пользователя. Блок «Начисления (расчёт)» показывает:

  • Личное — заработано с собственной рабочей ссылки
  • Реферальное — заработано с подписчиков своих рефералов
  • Начислено — сумма личного + реферального
  • Выведено — уже выплачено воркеру
  • Баланс — доступно к выводу (начислено минус выведено)

Более детальную статистику смотрите в Статистика → По пользователям и Дневные отчёты.

💸 Заявки на вывод

Когда воркер запрашивает вывод, заявка появляется в Выводы → Заявки на вывод.

1
Найдите заявку со статусом Pending
Все новые заявки отображаются первыми. Статус pending = ожидает обработки.
2
Проверьте данные
Убедитесь, что сумма не превышает баланс воркера. Проверьте реквизиты (метод вывода и детали платежа).
3
Одобрите или отклоните
В Telegram-боте (раздел администрирования) нажмите «Одобрить» или «Отклонить». При одобрении сумма автоматически вычтется из баланса воркера и воркер получит уведомление.

📢 Рассылки

Рассылки создаются через Telegram-бота (раздел администрирования). Здесь в Рассылки вы можете видеть статус отправки и логи доставки.

✅
Статус рассылки
Draft — создана, не запущена. Running — отправляется прямо сейчас. Done — завершена. Failed — ошибка.
📬
Логи доставки
В разделе «Логи доставки» можно посмотреть, кому дошло сообщение, а кому нет (например, если воркер заблокировал бота).

📅 Дневные отчёты

В разделе Статистика → Дневные отчёты хранятся записи о ежедневной работе. Каждый дневной отчёт содержит данные по клиенту: общее число заявок, ставку клиента, ссылку. На основе этих данных системa начисляет заработок воркерам.

💡
Пропущенные дни
Если отчёт за какой-то день не был внесён вовремя, он попадёт в раздел «Пропущенные дни». Данные всё равно можно внести задним числом — воркеры получат доначисление автоматически.
