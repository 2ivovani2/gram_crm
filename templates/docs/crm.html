Работа с CRM

CRM — это инструмент для ежедневного учёта финансов и заявок. Сюда вносятся данные по деньгам (Cash Flow) и количеству заявок, система автоматически считает итоги и формирует отчёты.

🔐 Вход в CRM

1
Откройте CRM
Перейдите по адресу /crm/ или нажмите ссылку «Открыть CRM» в боковом меню.
2
Войдите через Telegram
Нажмите кнопку «Войти через Telegram» — откроется окно подтверждения. Нажмите «Confirm» в Telegram. Вход произойдёт автоматически.
3
Попадёте на Dashboard
Главная страница показывает статус данных за сегодня, последние отчёты и прогресс недельного плана.
⚠️
Важно: нужен аккаунт в боте
Чтобы войти в CRM, ваш Telegram-аккаунт должен быть зарегистрирован в системе — то есть вы когда-либо запускали нашего бота. Если видите ошибку «не зарегистрированы» — сначала напишите боту /start.

🏢 Пространства (Workspace)

CRM поддерживает несколько рабочих пространств. Каждое пространство — это отдельный проект или клиент с независимыми данными, настройками и участниками.

🔄
Переключение пространства
Если у вас несколько пространств — в левой панели CRM рядом с названием появится стрелка. Нажмите — выберите нужное. Все данные и доступы переключатся автоматически.
👤
Роль в пространстве
В разных пространствах у вас могут быть разные роли. Ваша текущая роль отображается в левом нижнем углу рядом с именем.
➕
Создать пространство
Новые пространства создаются через эту Django-панель. Не забудьте после создания добавить себя как Owner.

👑 Роли в CRM

РольЧто может делать
Owner Всё: вносить данные, смотреть отчёты, управлять участниками, менять планы, экспортировать данные
Finance Только вносить финансовые данные (Cash Flow) — вкладка «Деньги»
Applications Только вносить данные по заявкам — вкладка «Заявки»
Viewer Только просматривать Dashboard (без финансовых деталей)

💵 Внесение данных: Cash Flow

Каждый день нужно вносить финансовые данные. Это делается через меню «Внести данные → Cash Flow».

1
Перейдите в Cash Flow
Нажмите «Cash Flow» в левом меню. По умолчанию откроется форма на сегодняшнюю дату. Для другого дня — выберите дату вверху формы.
2
Заполните поля
Поступления ($) — сумма всех входящих денег за день
Расходы ($) — сумма всех выплат и расходов
Заработок ПП ($) — заработок с партнёрских программ
Заработок Приват ($) — заработок с Привата
Скриншот с КБ — фото/скрин для подтверждения
3
Нажмите «Сохранить»
Данные сохранятся. Если в этот же день уже внесены данные по заявкам — система автоматически сформирует сводный отчёт.
💡
Редактирование задним числом
Данные можно вносить за прошлые даты. Просто выберите нужный день в поле «Дата» вверху формы — данные обновятся, отчёт пересчитается автоматически.

📋 Внесение данных: Заявки

Параллельно с финансами вносятся данные по заявкам через меню «Внести данные → Заявки».

🔢
Количество заявок
Число заявок, обработанных за день. Система автоматически считает средний заработок на одну заявку.
💸
Заработок с заявок ($)
Общая сумма, заработанная с обработки заявок за этот день.
📝
Примечания
Необязательное поле. Можно записать что-то важное о дне — особые события, аномалии, пояснения.

📈 Отчёты и история

Когда за один день внесены и финансы, и заявки — система автоматически формирует Сводный отчёт. Он включает все показатели за день плюс процент выполнения недельного плана.

Где смотреть отчёты

1
Dashboard — сегодня и последние 7 дней
На главной странице CRM сразу видно: внесены ли сегодня оба блока данных и последние отчёты.
2
История — за любой период
Меню «История» → выберите даты → таблица со всеми днями. Зелёный значок = данные внесены и отчёт готов. Жёлтый = данные частично внесены.
3
Детальный отчёт
Нажмите на дату в истории — откроется полный отчёт за день с разбивкой, процентами выполнения плана и готовым текстом для отправки.

📅 Недельные планы

В разделе Администрирование → Планы задаются плановые показатели на каждую неделю. Система сравнивает фактические данные с планом и показывает процент выполнения в каждом отчёте.

✅
Подсказка
Неделя в системе всегда начинается с понедельника. Если вы вводите план в середине недели — он будет применён к текущей и всем последующим отчётам за эту неделю.

📤 Экспорт в Excel

Меню «Экспорт» — выберите период дат и нажмите «Скачать». Файл Excel содержит все данные за выбранный период: финансы, заявки, балансы, проценты выполнения плана.

👥 Управление участниками

Раздел Администрирование → Участники (только Owner). Здесь можно:

  • Добавить нового участника по его Telegram ID
  • Изменить роль существующего участника
  • Отозвать доступ (участник перестанет видеть данные)
⚠️
Пользователь должен быть зарегистрирован
Добавить участника в CRM можно только если он уже запустил нашего Telegram-бота. Иначе его Telegram ID не будет в базе и система выдаст ошибку.
